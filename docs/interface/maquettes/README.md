# Maquettes — EcoRide

Ce dossier regroupe les maquettes (écrans) de l’interface EcoRide.

## Organisation

- `ecrans_rechercher_un_covoiturage/` : maquettes des écrans de recherche
- `ecrans_resultats_recherche/` : maquettes des résultats
- `ecrans_tableau_de_bord/` : maquettes des écrans de l’espace utilisateur / tableau de bord

Chaque sous-dossier contient les visuels associés à un parcours.
