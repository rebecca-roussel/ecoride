# Liens — Gestion de projet (EcoRide)

Ce dossier centralise les liens vers les ressources externes utilisées
pour organiser et piloter le projet EcoRide (suivi, cadrage, documentation).

Il ne contient pas de livrables techniques et ne remplace pas les fichiers
présents dans le dépôt. Il sert uniquement de point d’entrée vers des outils
externes lorsqu’ils sont nécessaires.

## Contenu

Chaque ressource est documentée dans un fichier Markdown dédié (ex. `miro.md`, `kanban_notion.md`).
Chaque fichier précise :

- l’usage de l’outil dans le projet,
- le lien d’accès,
- la date de dernière mise à jour,
- le niveau de visibilité pertinent.
